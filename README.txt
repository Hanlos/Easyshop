Easyshop v1.0a
-=-=-=-=-=-=-=
An E-Commerce solution for Plone v2.5-v4.0 supporting a shopping
basket, stock quantities, differing delivery and taxation rules
based on criteria, product variants and Paypal payment processing.

Please see http://wiki.github.com/ned14/Easyshop/whychooseeasyshop
for why we think that Easyshop is the BEST e-Commerce platform
available for Plone.

Please see http://wiki.github.com/ned14/Easyshop/starttoendtutorial
for how to install and configure.

Please see http://github.com/ned14/Easyshop/issues for known bugs
and to which you should report new bugs (or patches fixing them!).

Please see http://wiki.github.com/ned14/Easyshop/commercialsupport
for details on commercial support.
